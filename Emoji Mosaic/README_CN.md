# Emoji Mosaic

Emoji Mosaic 是一个非常有趣的项目，可以将图片转换成 Emoji 组成的马赛克图片，原项目可以在这里找到：https://github.com/ericandrewlewis/emoji-mosaic

你现在使用的是基于 JSBox 的离线版本，感谢原作者 [Eric Lewis](https://github.com/ericandrewlewis) 的付出！
