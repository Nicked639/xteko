# AnyKey

- 一个用于展示自定义键盘的样例脚本
- 同样也是一个高度可定制的万能键盘

# 小提醒

- 去设置界面把这个脚本设置成“键盘脚本”
- 在键盘使用时，选择文字，点击插件来运行
- 在主应用使用时，点击来运行或编辑插件
- 长按拖拽项目可以进行重新排序
