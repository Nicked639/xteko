# 微博热点
通知中心尽览实时微博热点

### 脚本特色
- 无需登录微博，无需打开 App

- 通知中心搜索预览微博

### 脚本功能与操作说明
- 左滑一条微博可以选择通过 App 打开该内容

- 点击用户头像会通过内建 Safari 浏览器打开微博，点按微博内容会通过 Webview 打开微博

- 顶部搜索栏通过输入关键字"$dh"或"$eh"来设定 Widget 默认与扩展高度（如：$dh 400）

- 「热搜」Tab 下可切换本地与全网微博，你也可以在「本地」Tab 下的搜索栏内输入城市或区县名称来自定义地名（请输入完整市或区名，如：「上海市」或者「海淀区」）

- 无论在脚本 App 内或者通知中心，你都可以直接搜索微博（注：弹出的键盘上已默认加载剪贴板内容，可以直接按确认搜索剪贴板）

- 点按微博列表右侧的图像或者播放按钮可以直接预览图像或者视频（注：图像较多时需要等待加载完毕才可预览）

- 「热门」Tab 下可以切换「简单」或者「详情」微博话题展示

- 通过微博详情内的「分享微博」按钮可以分享微博 URL 链接

- 若在 JSBox 启动器中使用，你可以通过长按任意一条微博来关闭脚本
